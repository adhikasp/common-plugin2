Spring Boot 2 Common Plugin
---------------------------

Add this dependency in your ```pom.xml```

```xml
<dependencies>
  ...
  <dependency>
   <groupId>com.blibli.oss</groupId>
   <artifactId>common-plugin2</artifactId>
   <version>...</version>
 </dependency>
 ...
</dependencies>
```
